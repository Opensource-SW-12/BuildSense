# BuildSense


# 프로젝트 소개

BuildSense AI는 사용자의 실제 컴퓨터 사용 패턴을 분석하여 사용자에게 적합한 조립 PC 사양을 추천하는 AI 기반 시스템입니다.

사용자의 PC 사용 데이터, 하드웨어 정보, 실행 중인 프로세스 정보 등을 바탕으로 CPU, GPU, RAM, SSD, HDD, 파워 등의 부품 추천 방향을 제시하는 것을 목표로 합니다.

# 프로젝트 목적

기존 조립 PC 추천 방식은 사용자가 직접 부품을 검색하고 선택해야 하는 경우가 많아 컴퓨터 지식이 부족한 사용자에게 진입 장벽이 높습니다.

본 프로젝트는 실제 사용 데이터를 기반으로 사용자의 사용 목적과 환경에 맞는 PC 구성을 추천하여 불필요한 과소비를 줄이고, 초보 사용자도 쉽게 조립 PC 사양을 결정할 수 있도록 돕는 것을 목표로 합니다.

# 주요 기능

- 사용자 PC 하드웨어 정보 수집
- CPU, RAM, GPU 사용량 모니터링
- 실행 중인 프로세스 정보 기록
- 사용 기간 설정 및 데이터 저장
- 사용자 컴퓨터 지식 수준 입력
- 추천 부품 선택 옵션 제공
- 분석 결과 기반 PC 부품 추천
- 추천 결과 확인용 UI 제공

# 데이터 사용 계획

본 프로젝트는 사용자의 PC 사용 패턴 분석뿐만 아니라, 필요한 부품 추천 데이터를 구축하기 위해 공개된 오픈소스 데이터와 외부 데이터를 활용할 예정입니다.

수집한 데이터는 엑셀 또는 CSV 형태로 정리한 뒤, 분석에 필요한 항목만 선별하여 전처리할 계획입니다. 추가로 필요한 정보는 크롤링을 통해 보완하고, 이를 바탕으로 추천에 활용할 데이터베이스를 구성할 예정입니다.

# 기술 스택

- GitHub
- Jira

# 현재 구현 상태

현재는 전체 프로그램 구조와 기본 실행 화면을 구성하는 단계입니다.
구현된 화면에는 데이터 수집 동의 화면, 사용자 설정 화면, 모니터링 실행 화면, 설정 요약 화면 등이 포함되어 있습니다. 향후 Tkinter Designer를 활용하여 UI를 개선하고, 분석 결과를 바탕으로 추천 결과 화면을 구현할 예정입니다.
